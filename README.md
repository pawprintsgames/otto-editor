## Development

Otto Editor is a web app built using [SvelteKit](https://svelte.dev/docs/kit/).

Make sure you have [NodeJS](https://nodejs.org/en) installed, then run:

```bash
npm install
```

Start a dev server:

```bash
npm run dev
```

To build the production version:

```bash
npm run build
```
